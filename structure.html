Course Schedule

Structure

Both versions of a course - individual and bootcamp - have the same number of class hours and cost exactly the same: each module (or “course”) runs 60 hours and costs $1,000. The bootcamp curriculum is developed to flow from one course to the next, and to develop the student’s competence, confidence and portfolio in a way that becomes gradually more demanding. In exceptional circumstances, a student can jump into the middle of a bootcamp for a given module. Please note that on your application and the CodeAbode Director will discuss that with you.

Upcoming Courses for Oct. 12 - Dec. 18:

Modern Web Design and Development with HTML5 & CSS3
Mon/Wed

This course will cover the essentials of modern web site design and development, using HTML5 and CSS3. Students will be exposed to the entire development pipeline, including early research and planning, wire framing, selecting a final design, and production. The focus of the course will be on developing strong foundational knowledge of “raw” HTML and CSS. From there, students can expand their skills using any of the popular web development frameworks.



Introduction to JavaScript Programming & Web Apps
Tues/Thurs

JavaScript (JS) is the language of interactivity on the web. This course is designed to teach programming and JS fundamentals, in service of creating web applications. We will start with basic programming & web concepts: storing data, making decisions, manipulating webpages and getting user input. Then we will move into using JS to make dynamic and interactive webpages.



* A single course can't hope to teach everything about HTML/CSS or JS. The most important skill you need in the technical world is the ability to problem solve and teach yourself. The lessons and assignments in these classes are structured around building that skill.
