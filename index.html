About


We are a new Chicago-based code academy, designed to turn students into web and mobile app developers. We offer both ‘Bootcamp’ and part-time options*.The Bootcamp runs for 14-weeks, Monday through Friday from 10AM to 5PM with one-hour for lunch. It includes 7-courses that build upon each other, each lasting approximately two weeks.The part-time program is held twice a week, for 10 weeks – from 6PM to 9PM. Classes will meet on either Monday and Wednesday evenings or Tuesday and Thursday evenings. *(See our Calendar for class schedule and start dates).

CodeAbode is a division of the National Museum of Health and Medicine in Chicago (NMHM- Chicago). Their mission is to promote and encourage scientific research, innovation, and knowledge of health and medicine. Along with engaging the public through cutting-edge technologies and digital exhibitions, they serve as a center of education for scholars, innovators, students, educators and the general public. As part of the museum, CodeAbode's niche in the marketplace will be our focus on health, medicine and fitness applications. Students do not need to have a health or medicine background or concentration, but many of our classroom examples will be from that genre.

Structure


At CodeAbode, we believe that you learn by doing, especially when it comes to designing and programming mobile apps. We dedicate many classroom hours to actual programming in a lab environment. Our instructors are developers with a passion to teach and our bootcamp schedule includes an hour-long midday break, where students can eat lunch and network with other students or their instructors.

Each student can design an app to their own specs- be it a game, product idea, or a mobile version of software that they already use. But, we will also have apps that you can build based on our core mission of health and medicine. We can provide different apps for you to clone and work on including, a fitness app, a biomedical visualization app, or a mobile analytics app based on data from an EHR system.

Each of you will be placed in a team so that you can learn from your fellow students as well as the instructors. Each person will take on different roles so that you are learning the techniques as deeply as possible. Every class larger than 20-students will have assistant instructors as well as the lead instructor to insure that everyone is able to receive the attention that they need.

All classes will be taught at the National Museum of Health and Medicine- Chicago, so at breaks you can mingle in the exhibits or the auditorium space, and network with the interns or staff of the Museum, as well as with your fellow classmates.

Courses

Learn in bootcamp or night classes.


HTML and CSS3

Here you will learn how to set up a web site using the latest generation of HTML (hypertext markup language) and CSS (cascading style sheets). These are the foundational authoring tools for the web, supported by all modern web browsers. We will have sample web sites for you to author, and through the class you will begin to build a portfolio of your work. The basics are quite easy to master, and with our help you will be able to create dynamic web sites to show off your newly-learned skills.



Phone Apps/Introduction to Programming

This course will focus on getting you ready to really program, by teaching the basics of programming using JavaScript. This is a language that much mobile development uses, especially for accessing databases that you will continue to learn and extend across the sequence. We will also be introducing some tools that allow you to create once, and publish apps across platforms. In addition to supercharging your portfolio right away and getting you excited about how fast you are creating meaningful apps that work both on the web and mobile devices, you will expand your understanding of many of the concepts you will master at a deeper level in future courses. At the end of this module, you’ll be able to publish on the AppStore and GooglePlay


Server-side Integration

This course is where you will start to focus on where many apps find their real value – integrating with existing data bases and being able to tap into the power of servers that can be located anywhere in the world. You got a taste of this already in the Mobile Web Apps course, but now we are going to add the dynamics of large data sets.You will learn how to access and control remote databases, how to selectively pull and update data, how to use APIs that are built into application software, and how to optimize the use of both proprietary and public databases over the network. This is a critical skill for working in a corporate IT department, and a powerful tool for building apps that have value in the real world.



Advanced Programming/Introduction to Object-Oriented Programming

This course will extend the second class. Now that we have integrated the server-side, we are ready to teach you dynamic updating of your apps, client-side calculations and visualizations, extensions to the auto-creation of iOS and Android apps, and much more. In this course we will also introduce you to object-oriented programming concepts because you will need it to take full advantage of the native languages of the top two phone environments. The language we’ll introduce you to for OOP is Java.

Phone-specific Programming


Note: For the bootcamp, each student can pick either Apple or Android to learn. We want to keep the number of languages you have to learn limited, so we are giving you an option of one or the other. If you want to learn both, you can take the other one at night. Option 1: Native iOS - Swift Having already built some Apple mobile apps in the Mobile Web Apps course, in this course you will dive in more deeply and learn to build fully-native iPhone and iPad apps using a new Apple tool called Swift. Swift is an easy-to-learn successor to the C and Objective C languages and generates superfast and efficient code to run on iPhones and iPads. You will learn to code in Swift and XCode, how they compare to HTML, and how to not only design apps, but also to make them dynamic, even using features of the devices like the accelerometer and sensors. Option 2: Native Android - Java Next you are going to learn how to extend your newly-learned native-app skills to the world of Android mobile devices, using the Java, programming language. In this course, we will use Java the Android Developer Tools, and the Integrated Development Environment (IDE) to develop native Android apps. You will build on what you learned in the previous courses, and the comparisons and contrasts will start coming faster as you do similar things in what will now be your fourth application environment. In this class however, we won’t just be recreating the same app again. We want to build your portfolio, so we’ll create new apps, and then you may even choose to take the challenge of converting them back to iOS as well.

Advanced Server and Databases


This course is where you will start to focus on where many apps find their real value – integrating with existing data bases and being able to tap into the power of servers that can be located anywhere in the world. You got a taste of this already in the Mobile Web Apps course, but now we are going to add the dynamics of large data sets. The most widely used programming language for integrating remote data resources into mobile apps is JavaScript. You will learn how to access and control remote databases, how to selectively pull and update data, how to use APIs that are built into application software, and how to optimize the use of both proprietary and public databases over the network. This is a critical skill for working in a corporate IT department, and a powerful tool for building apps that have value in the real world.
